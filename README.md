## 📌 About the Project

KVK: King VS King is a modern gaming landing page for a mobile strategy game focused on kingdom development, army building, and PvP battles between players.

The project is built as a multi-section landing page with smooth navigation, interactive UI elements, and a strong visual presentation of the game concept.

The main goal is to promote the game and showcase its core gameplay features in an engaging way.

---

## 🧩 Structure of the Project

The landing page consists of modular HTML sections:

- Header (including burger menu for mobile navigation)
- Hero
- Concept
- How to Play
- Core Features
- Commanders (implemented with Swiper.js)
- Gallery
- Player Reviews
- FAQ
- Footer
- Legal pages:
  - Privacy Policy
  - Terms & Conditions
